# filterax Issue Drafts

These drafts mirror the grouped-review format used for `pyrox/issues`: a small
set of wave-level markdown files, each carrying enough design context that
another agent could implement the backlog without reopening the private design
docs.

Important naming note:

- The source docs still use `ekalmX` / `filterX`.
- This draft set freezes the repo and package identity as `filterax`.
- When a source-path reference still points into `design_docs/filterX/...`, that
  is only a source reference, not the intended package name.
